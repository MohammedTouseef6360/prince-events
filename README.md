# PRINCE EVENTS

Royal wedding snacks and starters website. No database required — works out of the box.

## Requirements

- **Node.js 18+** (Download from https://nodejs.org)

## Setup

```
npm install
```

## Run

| Platform | Command |
|----------|---------|
| **Windows** | Double-click `start.bat` |
| **Windows (phone access)** | Double-click `start-mobile.bat` |
| **Mac/Linux** | `bash start.sh` |
| **Mac/Linux (phone access)** | `bash start-mobile.sh` |
| **Any (terminal)** | `npm run dev` |

Open http://localhost:3000

## Admin Panel

- URL: http://localhost:3000/admin/login
- Password: `prince@123`

## Build for Production

```
npx next build
npx next start
```

Or double-click `start-prod.bat`
